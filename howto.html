How to test ATs

Virtual Machines

To test across multiple operating systems, a virtual machine monitor may be required. If this is the case, download and run the VirtualBox installer.

To test Mac OS from another operating system, review this guide to installing Mac OS as a Virtual Machine.

To test Windows from another operating system, download one of the many free Windows virtual machines from Microsoft.

Browsers

To test Safari for Mac OS, run the pre-installed application.

To test Edge or Internet Explorer for Windows, run one of the pre-installed applications.

To test Chrome from any operating system, download the Chrome installer.

To test Firefox for any operating system, download the Firefox installer.

To test Opera for any operating system, download the Opera installer.

Assistive Technologies

To test VoiceOver for Mac OS, run the pre-installed application.

To begin using VoiceOver on Mac OS, press Command + F5.

When using VoiceOver with Safari, you can enter the page by pressing Control + Options + Shift + Down. This will take you to the first readable item. To move around the page, hold Control + Option and press the left or right arrow keys. To move between individual words, hold Option and press the left or right arrow keys.

When using VoiceOver with Chrome, you will not be able to navigate through the page using modifier keys, but the page should be read aloud to you.

To test with JAWS for Windows, download and run the JAWS installer.

To test with NVDA for Windows, download and run the NVDA installer.

To test with Windows Eyes for Windows, download and run the Windows Eyes installer.

When using JAWS, NVDA, or Windows Eyes, load the page and it should be read aloud to you.
